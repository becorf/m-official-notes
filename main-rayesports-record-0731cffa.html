背景 · 竞技服务的定位

雷竞技 是一个以电子竞技为核心参照的信息与互动平台，围绕赛事数据、选手动态及对战趋势提供结构化内容。与泛娱乐平台不同，它更侧重竞技本身的逻辑——赛程、队伍构成、历史交锋等维度的整理，帮助用户快速建立对赛事的全景认知。

平台设计上强调信息密度与低干扰，界面节奏偏向工具属性，适合需要快速查阅竞技数据的用户。以下从访问准备、核心功能、常见场景三个层面展开。

访问指南 · 关键前提

使用 雷竞技 前建议确认以下三点：网络环境稳定，设备浏览器保持更新（Chrome / Edge / Safari 近年版本）；首次访问时可留意页面语言与区域选项，部分赛事数据以 UTC 时间标注。平台无需注册即可浏览大部分公开内容，但若需要订阅特定战队或参与社区讨论，则需完成账号创建。

界面导航以赛事分类为主轴，左侧栏列出主流项目（如 Dota 2、CS2、VALORANT、LoL 等），右侧面板展示即时比分与即将开始的场次。数据更新频率通常在 30–60 秒区间，关键对局会附带简短文字复盘。

常见场景与使用提示

日常使用中，用户多用于赛前查看双方近期交锋记录、地图胜率以及选手个人数据。例如，在大赛前对比两支队伍的过往交手战绩，或观察特定选手在近十场中的击杀/死亡/助攻曲线。这些信息在 雷竞技 的「对战详情」与「选手档案」模块中可直接调取。

另外，平台提供简易的赛程日历导出功能（ICS 格式），适合需要将赛程同步至个人日历的用户。需注意：部分深度统计（如实时经济差、视野得分）仅在大型锦标赛期间开放完整维度，常规赛事可能仅保留基础数据。

  • 赛事列表按日期排序，支持筛选特定阶段（小组赛 / 淘汰赛）
  • 选手页面展示近 20 场表现热力图，含 KDA、分均伤害、参团率
  • 队伍页面含阵容变动时间轴，标注转会窗口与替补记录

关于信息可靠性

数据来源标注于各页面底部，主要采自官方赛事 API 及合作数据商。历史记录保留最近三个赛季，部分早期赛事因数据口径调整可能存在 ±2% 的浮动，页面会有相应备注。

若需直接了解当前赛季赛程、战队 roster 或选手统计，可通过 雷竞技 首页进入对应模块。平台无强制弹窗或自动跳转，所有导航均为手动触发。
